  1. HTML 2.0

    November 24th, 1995

    HTML 2.0 was published as IETF RFC 1866. Includes basic capabilities for tables, forms, and images.

  2. HTML 3.2

    January 14th, 1997

    W3C publishes the HTML 3.2 Recommendation. Extended doctype and script standards. Influenced mainly by IBM, Microsoft, and Netscape.

  3. HTML 4.0

    December 18th, 1997

    W3C publishes HTML 4.0. Significant changes with several new elements such as <button>, <label>, and <span>. Sought to phase out Netscape's visual markup features in favor of CSS.

    The following year, the W3C DOM Working Group also publishes a complete model for HTML, coined "DOM Level 1".

  4. HTML5

    January, 2008

    W3C publishes HTML5 as working draft. Largely improves language support for multimedia and introduces lots of new markup with Javascript APIs. <canvas>, <video>, and <audio> elements added. Also, new semantic elements for page structure, like <section>, <article>, <header>, <footer>, and <nav> added.

  5. Shadow DOM

    May 22, 2012

    First working draft for Shadow DOM introduced. Embraced by Google & the Chromium team.

  6. HTML Templates

    Feb 14, 2013

    First working draft for HTML Templates. Sponsored by Google/Chromium team and Microsoft, and other community contributors.

  7. Custom Elements & HTML Imports

    May 14, 2013

    W3C publishes first working draft (v0) of Custom Element's spec. Another spec and working draft is introduced for HTML Imports.

  8. X-Tag

    June 13, 2013

    Microsoft introduces the X-Tag project, a new library for building custom elements.

  9. Polymer 0.1.0

    November 26th, 2013

    Polymer project released to public. Shadow DOM and Custom Elements have a framework at Google.

  10. Polymer 1.0 release

    May 29th, 2015

    Polymer library is re-written from ground up. It's 3 to 4X faster and has 36% less code. Element product lines paper-elements, iron-elements, google-web-components, platinum-elements, and gold-elements introduced.

  11. Custom Elements v1

    January 19th, 2016

    W3C releases updated V1 specification of Custom Elements spec. Shortly later, the webcomponents.js polyfill is updated for v1.

  12. Progressive Web Apps

    May 19th, 2016

    Progressive Web Apps demonstrated at Google IO. The P.R.P.L pattern with HTML Imports, HTTP/2, & Service Worker discussed.
